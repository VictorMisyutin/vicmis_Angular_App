COLLEGE BASKETBALL API DOCUMENTATION

vicmis.com/api/cbb/get-player/<player name>

Query the "CBB" CSV for specific player. Returns
Points, Assists, Rebounds, Pts+Asts, Rebs+Asts,
Reb+Asts, Pts+Reb+Ast.

vicmis.com/api/cbb/all-players

Query the "CBB" CSV for all players.
Returns all players and their stats.
