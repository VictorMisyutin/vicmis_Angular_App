An Introduction to Vim for Beginners

Victor Misyutin

My first introduction to terminal text editors was using Nano to help set up my Ubuntu EC2 intance. The only reason I used Nano was because that was the text editor that the guy in the video was using.

It wasn't until about a month later that I even knew there were alternative to Nano. Mainly, Vim.

However, I never wanted to use Vim because unlike Nano, it seemed far more complex and difficult to use. I remember the first time I tried to use Vim instead of Nano I could not even figure out how to write to the file. For that reason I used nano for about four months.

Then I watched some random YouTube video where some guy was using Vim and was literally making magic on my screen. He knew like every shortcut and command and made Vim look extremly powerful. Many of those commands and shortcuts are not even possible using Nano.

Then I started learning Vim and haven't looked back.

What is Vim

Vim and Nano are both in-terminal text editors. Since they are terminal based that means that you can only interact with them using your keyboard, no mouse inputs. This is quite annoying but they are very useful when you SSH onto another system and edit and read files on that system.
