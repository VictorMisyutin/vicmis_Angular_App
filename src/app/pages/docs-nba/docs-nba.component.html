NBA API DOCUMENTATION

vicmis.com/api/nba/get-player/<player name>

Query the "NBA" CSV for specific player. Returns
Points, Assists, Rebounds, Pts+Rebs+Asts, Pts+Rebs, Pts+Asts, Steals, Rebs+Asts, Blks+Stls, Turnovers.

vicmis.com/api/nba/all-players

Query the "NBA" CSV for all players.
Returns all players and their stats.

vicmis.com/api/nba/bot-output/get-player/<player name>

Query the "bot-output" CSV for specific player. Returns
Assists, Points, Rebounds, Reb+Ast, Pts+Reb+Ast.

vicmis.com/api/nba/bot-output/all-players

Query the "bot-output" CSV for all players.
Returns all players and their stats.

vicmis.com/api/nba/diff/get-player/<player name>

Query the "differences" CSV for specific player. Returns
Assists, Points, Rebounds, Reb+Ast, Pts+Reb+Ast.

vicmis.com/api/nba/diff/all-players

Query the "differences" CSV for all players.
Returns all players and their stats.
