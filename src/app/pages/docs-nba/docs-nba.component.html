NBA API DOCUMENTATION

vicmis.com/api/bot-output/get-player/<player name>

Query the "bot-output" CSV for specific player. Returns
Assists, Points, Rebounds, Reb+Ast, Pts+Reb+Ast

vicmis.com/api/bot-output/all-players

Query the "bot-output" CSV for all players.
Returns all players and their stats

vicmis.com/api/diff/get-player/<player name>

Query the "differences" CSV for specific player. Returns
Assists, Points, Rebounds, Reb+Ast, Pts+Reb+Ast

vicmis.com/api/diff/all-players

Query the "differences" CSV for all players.
Returns all players and their stats
